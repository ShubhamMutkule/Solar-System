# 🌞 Solar System

A modern **Solar Energy Management Web Application** built with Django.  
It provides solar solutions information and allows users to send inquiries through a contact form.

---

## 🚀 Features

- 🌱 Modern responsive UI design  
- ☀️ Solar energy services showcase  
- 📩 Contact form with database storage  
- 👨‍💼 Django Admin panel for managing inquiries  
- 📊 Clean backend structure  
- 🔐 Secure Django framework  
- ⚡ Fast and lightweight project  

---

## 🛠️ Tech Stack

- Python 🐍  
- Django 🌿  
- HTML5  
- CSS3  
- SQLite (default database)

---
## 📩 Contact Form Fields

Users can submit inquiries using the contact form with the following fields:

- 👤 Name  
- 📧 Email  
- 📞 Phone (Optional)  
- 🏙️ City (Optional)  
- ⚡ Service Type (Optional)  
- 💬 Message  

All submitted data is securely stored in the Django Admin panel for management and review.

---

## 🧑‍💼 Admin Panel Features

The Django admin panel provides full control over all contact submissions:

- 📋 View all contact messages in a structured table  
- 🔍 Search inquiries by name, email, phone, or city  
- 🎯 Filter messages by service type (Home, Business, Industrial, etc.)  
- 📅 Track submission date and time  
- 🗂️ Manage and organize leads easily  
- 📊 Monitor customer inquiries efficiently  

This helps in converting inquiries into real business leads quickly and effectively.
